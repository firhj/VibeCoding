## 一、Prompt 正文（粘贴到 TRAE Work）

```
请使用 frontend-design 技能，帮我创建一个闯关式 AI 编程入门互动学习网站，生成为单个 index.html 文件。

## 设计方向
- **风格**：深色科技风，宇宙/星空主题
- **主色调**：深蓝底色 (#0a0a1a → #1a1a3e)，亮紫色点缀 (#a78bfa)、青色强调 (#22d3ee)
- **字体**：标题用 Orbitron 或 Rajdhani（科技感），正文用 Noto Sans SC（中文友好），避免 Inter/Roboto/Arial
- **氛围**：粒子星空背景、微光晕、毛玻璃卡片、柔和的霓虹边框

## 模块要求（必须完全解耦，互不影响）
每个模块用独立 CSS 类和 JS 命名空间，避免全局污染：

### 模块1：首页/Landing（.module-homepage）
- 大标题"AI编程入门9关"，副标题"从零到一的AI开发之旅"
- 3 句简介文案 + CTA 按钮"开始挑战"
- 粒子星空的 canvas 背景（命名空间：HomepageBg）

### 模块2：关卡选择器（.module-level-selector）
- 3x3 九宫格布局，每格代表一关
- 关卡锁定/解锁状态（通过第0关后逐关解锁）
- 已通关显示 ✓，当前可挑战显示脉冲动画，未解锁显示锁定图标
- 每格显示：关卡编号、图标emoji、简短标题
- 关卡列表：
  第1关：🧠 AI大脑的训练
  第2关：🦾 AI的手和脚
  第3关：💻 AI编程环境
  第4关：🎯 指挥AI工作
  第5关：⚡ 现代AI工具
  第6关：🛡️ AI的安全课
  第7关：🔍 AI代码审查
  第8关：🏗️ AI自动建应用
  第9关：🚀 AI运维

### 模块3：内容展示区（.module-content-display）
- 独立滚动容器，展示当前关卡的讲解内容
- 内容分3段，每段有标题、正文、可视化占位区
- 段与段之间有平滑的淡入动画
- 支持"大白话讲解"和"可视化演示"两种子视图切换

### 模块4：动画演示区（.module-animation-stage）
- 每关专属的简易交互动画
- 第1关：拖拽 token 看 AI 预测概率
- 第2关：M×N vs M+N 连线对比动画
- 第3关：同步 vs 异步代理时间线对比
- 第4关：输入模糊/精确指令，看 AI 输出差异
- 第5关：工具进化时间线滑动条
- 第6关：提示注入攻击演示
- 第7关：雷达图对比 AI vs 人类擅长领域
- 第8关：传统开发 vs AI 开发时间线对比
- 第9关：运维模式演变动画

### 模块5：动手实验区（.module-experiment）
- 每关一个交互式小实验
- 第1关：比较 9.1 vs 9.9 和奥数题，看 AI 表现差异
- 第2关：有数据 vs 无数据时 AI 的回答差异
- 第3关：精确指令 vs 模糊指令的输出对比
- 第4关：有无上下文时 AI 的输出质量对比
- 第5关：战略模式 vs YOLO 模式的对比
- 第6关：在文本中找出隐藏的恶意提示
- 第7关：AI 审查一段有 bug 的代码
- 第8关：10秒内生成 Todo 应用
- 第9关：AI 诊断 vs 人类诊断时间对比

### 模块6：测验组件（.module-quiz）
- 3 道选择题，每题4个选项
- 选对：绿色闪光 + ✓ 标记
- 选错：红色抖动 + 显示正确答案
- 全部答对弹出"通过"动画（粒子爆炸效果）
- 答错可重试，不限次数
- 通过后解锁下一关

### 模块7：进度条（.module-progress-bar）
- 顶部固定，显示 0/9 → 9/9 进度
- 每过一关节点亮起并连接
- 带百分比数字

### 模块8：毕业页面（.module-graduation）
- 9关全部通过后触发
- 毕业证书样式卡片
- 9个核心概念回顾
- 粒子烟花庆祝动画
- "下一步"指引区

## 交互要求
- 所有动画使用 CSS animation/transition，避免 JS 定时器动画
- 卡片 hover 有光晕效果（box-shadow 过渡）
- 页面切换用淡入淡出（opacity 过渡，300ms）
- 测验交互流畅，选错时温和抖动（translateX ±5px, 3次）
- 移动端竖屏适配（768px 以下切换为垂直布局，九宫格变为列表）

## 技术约束
- 单个 index.html，内联 CSS 和 JS，不需要额外文件
- CSS 变量管理所有颜色/间距/字体
- 每个模块独立命名空间，零全局污染
- 不使用任何外部 CDN 依赖（字体用 CSS @font-face 或系统后备）
- 兼容 Chrome/Firefox/Safari/Edge 最新版

## 性能要求
- 首屏加载 < 1s（粒子系统不阻塞渲染）
- 动画帧率 ≥ 30fps
- 移动端粒子数量减半
- 图片/图标使用 CSS/SVG 绘制，不使用外部图片
```
