# CS146S AI 编程互动学习网站

> 斯坦福大学 CS146S 课程 · 闯关式互动学习 · 零基础友好

基于 Stanford CS146S "AI With Ryan" 播客系列，将 9 周课程内容转化为中文互动网页，让你像打游戏一样从零学会 AI 编程。

---

## 为什么做这个？

斯坦福 CS146S 是一门面向所有人的 AI 编程入门课，但原版视频是英文的，且学习节奏较散。我们把它整理成 **9 关闯关式互动网页**——每关一个核心主题，配合课程要点、思考题和实践建议，帮你系统性地掌握 AI 编程全貌。

## 9 关学习路径

| 关卡 | 主题 | 一句话理解 |
|:---:|------|-----------|
| 1 | 大语言模型与 AI 开发入门 | AI 大脑是怎么训练出来的 |
| 2 | 把大语言模型变成自主执行者 | 给 AI 装上手脚（MCP 协议） |
| 3 | AI IDE | 选一个顺手的 AI 编程环境 |
| 4 | 编程代理模式 | 让 AI 自己写代码、自己跑测试 |
| 5 | 现代 AI 终端 | 用命令行指挥 AI 干活 |
| 6 | AI 测试与安全 | 怎么防止 AI 搞破坏 |
| 7 | 现代软件支持 | AI 写的代码要不要 review |
| 8 | 自动化 UI 和应用构建 | 让 AI 帮你做网页和 App |
| 9 | 代理部署后运维 | AI 上线了谁来管 |

```
基础认知 → 工具连接 → 开发环境 → 工作模式 → 技术加速 → 安全保障 → 代码质量 → 产品构建 → 部署运维
   第1关      第2关       第3关        第4关        第5关        第6关        第7关        第8关       第9关
```

## 课程亮点

- **闯关式设计** — 完成一关解锁下一关，进度可视化
- **零基础友好** — 用日常比喻解释技术概念，不堆术语
- **中英双语** — 原版英文字幕 + 中文翻译对照
- **单文件部署** — 一个 `index.html` 搞定，打开浏览器就能学

## 快速开始

```bash
# 克隆仓库
git clone <your-repo-url>

# 用浏览器打开
open CS146S/index.html
```

无需安装任何依赖，直接打开 HTML 文件即可开始学习。
